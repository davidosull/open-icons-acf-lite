# Changelog

All notable changes to ACF Open Icons - Lite will be documented in this file.

## [1.0.0] - 2024-12-17

### Added
- Initial release
- 300+ Heroicons (outline style)
- Colour palette support with 3 customisable tokens
- SVG caching for improved performance
- Clean, sanitised SVG output
- Helper function `acf_open_icon()` for theme templates
- Settings page under ACF menu
- WordPress.org plugin repository compatibility
